Simple Python Autoclicker
A lightweight and customizable Autoclicker built with Python using pyautogui and keyboard. This tool allows users to choose the click type (Left/Right), set a specific click interval, and define a custom hotkey to start and stop the clicking process instantly.

### Features
Custom Hotkey: Define any key to toggle the clicking on/off.

Toggle Logic: No need to hold the key down; press once to start, press again to pause.

Variable Interval: Set the delay between clicks in seconds (e.g., 0.1 for 10 clicks per second).

Click Choice: Supports both Left and Right mouse buttons.

Emergency Stop: Press ESC at any time to terminate the script.
